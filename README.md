# Expense_calculator_CRUD

A complete CRUD application for tracking income and expenses with modern UI and full functionality.

✨ Features
✅ Full CRUD - Create, Read, Update, Delete transactions

✅ 3 Filters - All - Income - Expense (radio buttons)

✅ LocalStorage - Data persists across browser sessions

✅ Live Totals - Income - Expense - Net Balance (color-coded)

✅ Responsive Design - Mobile + Desktop optimized

✅ Edit Mode - Click Edit → Modify → Update

✅ Clean UI - Modern gradients, animations, hover effects


USAGE :

1. Enter Description + Amount
2. Select Income/Expense (radio filter)
3. Click "Add Transaction"
4. Edit: Click "Edit" → Update form → "Update Transaction"  
5. Delete: Click "Delete" → Confirm
6. Reset: Clear form anytime


TECH STACK :Frontend: HTML5, CSS3 (Flexbox/Grid),  JavaScript
Features: LocalStorage, Event Delegation, Responsive Design

TEST EXAMPLE:

1. Add Salary ₹5000 (income) → incomeTotal = 5000
2. Add Food ₹1200 (expense) → expenseTotal = 1200  
3. Net Balance = 5000 - 1200 = ₹3800
